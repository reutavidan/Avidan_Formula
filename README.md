# AVIDAN FORMULA - Pediatric IOL Calculator

AI-based intraocular lens (IOL) power prediction for pediatric cataract surgery.

## Model Performance
- **MAE**: 0.39 D
- **Within +/-1.0 D**: 96.1%
- **R-squared**: 0.987

## Usage
Run locally:
```bash
pip install -r requirements.txt
streamlit run iol_formula_comparison_gui.py
```
